# Lumeflow SDK

Customer-facing SDK and runtime documentation for Lumeflow.

This repository is the documentation entrypoint for installing and validating Lumeflow local runtime environments from published package registries.

## Quick Start

If you are getting started for the first time, begin here:

- [Getting Started](./documentation/lumeflow/GETTING_STARTED.md)

That guide covers:

- Supported operating systems (Ubuntu Jammy and Noble).
- Adding apt keys and package sources.
- Installing `lumeflow-local-runtime`.
- Optional smoke-test install and execution.
- Troubleshooting common setup issues.
